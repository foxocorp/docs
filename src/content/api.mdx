# API

## Endpoints
- `https://api.foxochat.aop` - Production
- `https://api-dev.foxochat.app` - Development

## Error codes
### User errors
- `100` - Not found
- `101` - Email not verified
- `102` - Credentials duplicate
- `103` - Credentials is invalid
- `104` - Unauthorized

### Channel errors
- `200` - Not found
- `201` - Already exist

### Member errors
- `300` - Not found
- `301` - Already exist
- `302` - Missing permissions

### Message errors
- `400` - Not found

### Code errors (Email verification code)
- `500` - Is invalid
- `501` - Expired
- `502` - Wait to resend

### CDN errors
- `700` - Upload failed
- `701` - Invalid file format

### API errors
- `800` - Rate limit exceeded
- `801` - Empty body
- `802` - Validation error
- `803` - Route not found

### Unknown error
- `900` - Error
